Skyscouter Drone BBox Review

Draw or correct the box around the drone on the clean frame. The draft frame is only a starting point and may be wrong.

Rows marked draft are ignored by evaluation. Mark usable drone boxes as ok; mark ambiguous rows as skip; use negative only when there is no drone target in the frame.

How to correct: drag a new box on either image. The left clean image and the right draft image both update the same x/y values.
OK Drone: use when the box tightly encloses the visible drone and you are confident.
Occluded OK: use when the drone is partly hidden or cropped but you can still box the visible/inferable drone.
Negative: use only if this frame has no drone target or is a hard false-positive frame. It clears the box.
Skip: use when you cannot confidently label the frame. Skipped rows are ignored.
